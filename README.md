# 🐙 GitHub for Vibe Coders

Welcome! This repo contains all student and instructor resources for the **GitHub for Vibe Coders** workshop series.

The series is split into two sessions — you can attend one or both:

---

## 📚 Workshop 1 — Introduction

**Duration:** ~60 minutes | **Level:** Complete beginner

**What you'll learn:** What GitHub is, how to set up authentication, create your first repo, and make your first open source contribution.
