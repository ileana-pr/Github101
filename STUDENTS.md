## How to Add Your Name

1. **Fork** this repo (button in the top right)
2. **Clone** your fork: `git clone git@github.com:YOUR-USERNAME/github101.git`
3. **Create a branch**: `git checkout -b add-my-name`
4. **Edit this file** — add a row to the table below
5. **Push**: `git add . && git commit -m "Add [your name] to student list" && git push origin add-my-name`
6. **Open a Pull Request** on GitHub → Compare & pull request → Submit!
